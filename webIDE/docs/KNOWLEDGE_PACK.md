# Knowledge Pack

The default pack includes curated metadata and retrieval chunks aligned with:
- Qwen docs/cards
- The Stack v2
- CodeSearchNet
- OpenCodeInstruct
- DevDocs
- SWE-bench
- BigCodeBench
- EvalPlus / HumanEval+
- LiveCodeBench

Current implementation ships a minimal lexical retriever and manifest.
Future iterations can add vector embeddings and signed bundle updates.
